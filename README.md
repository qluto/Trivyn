# Triskly

**Sharpen your days quietly with three focuses**

Triskly is a cross-platform desktop app for practicing the "Three Wins" method (Rule of 3) proposed by J.D. Meier.

## Features

- **Three Goals** - Set 3 goals each for today, this week, and this month
- **Menu Bar Resident** - One-click access anytime
- **Floating Window** - Always visible on screen like Stickies
- **Achievement Animation** - Confetti animation when completing goals
- **Reflection Feature** - Support continuous improvement through daily reflection

## Requirements

- macOS, Windows, or Linux

## Installation

1. Download the latest release from [Releases](../../releases)
2. Install the application for your platform
3. Launch the app

## Usage

1. Launch the app and a triangle icon appears in the menu bar
2. Click the icon to open the popover and set your goals
3. Use "Show Floating Window" for always-on-top mode
4. Click the check button when you complete a goal
