# Tria - アプリ企画書

## 概要

| 項目 | 内容 |
|------|------|
| アプリ名 | **Tria**（トリア） |
| 名前の由来 | ラテン語で「3」 |
| プラットフォーム | macOS（常駐アプリ） |
| コンセプト | 3のルールで日々のフォーカスを研ぎ澄ます |

---

## コンセプト

Triaは、J.D. Meierが提唱した**"Three Wins"メソッド**（3のルール）を実践するためのmacOS常駐アプリです。

### 3のルールとは

> 「今日達成したい3つのこと」「今週達成したい3つのこと」「今月達成したい3つのこと」を設定し、定期的に振り返ることで、目の前のタスクに集中しながら大きな目標との整合性を保つ生産性メソッド。

### なぜTriaか

- **シンプルさ**: タスクを3つに絞ることで、本当に重要なことにフォーカス
- **階層性**: 日→週→月の目標を紐づけ、短期と長期の整合性を維持
- **振り返り**: 毎日の振り返りで、メタ認知と継続的改善を促進
- **常駐**: メニューバーに常駐し、いつでもアクセス可能

## 核心機能

### 1. 3つの目標設定

| レベル | 内容 |
|--------|------|
| 今日の3つ | 今日達成したいこと |
| 今週の3つ | 今週達成したいこと |
| 今月の3つ | 今月達成したいこと |

- 各レベル間の紐づけは**任意**で設定可能
- シンプルに使いたい人は独立管理、構造化したい人は階層リンク

### 2. メニューバー常駐

- macOSメニューバーに常駐
- クリックで現在の日次目標をクイック表示
- 控えめだが、いつでもアクセス可能
- ポップオーバーで目標の確認・編集

### 3. フローティングウィンドウ（Stickies風）

- macOS Stickiesのように画面上に常時表示可能なウィンドウ
- デスクトップの好きな位置に配置
- 常に最前面に表示（オプション）
- 半透明表示でコンテンツを邪魔しない
- メニューバーからワンクリックで表示/非表示を切り替え

### 4. ワンクリック達成 & 称賛演出

- 各目標の横にチェックボタン
- ワンクリックでDone（達成）ステータスに変更
- 達成時に**Confetti（紙吹雪）アニメーション**で称賛
  - ささやかだが達成感を演出
  - 小さな成功体験の積み重ねをサポート
- 達成済み目標は取り消し線で表示

### 5. 毎日の振り返り

- 1日の終わり（設定可能な時刻）に振り返りを促す通知
- 各目標の達成/未達成を記録
- 簡単なコメントを残せる（任意）

### 6. 履歴・アーカイブ

- 過去の目標と振り返りの閲覧
- 達成パターンの可視化（どのような目標が達成されやすいか）
- 継続の記録

## デザイン方針

### アイコン

- **モチーフ**: 三角形（3の象徴）
- **スタイル**: ミニマル、幾何学的
- **メニューバー**: 16x16 / 32x32（Retina）対応

### カラーパレット

| 用途 | 色 |
|------|-----|
| プライマリ | ネイビー / ダークブルー |
| アクセント | シアン / ライトブルー |
| 背景 | システム準拠（ライト/ダーク対応） |

### トーン

- **静謐**: 邪魔にならない存在感
- **知的**: 落ち着いた配色と余白
- **ミニマル**: 必要最小限の情報表示

## ユーザー体験フロー

```
朝
├── 通知: 「今日の3つを設定しましょう」
├── メニューバーから目標を入力
└── フローティングウィンドウを表示して作業開始

日中
├── フローティングウィンドウで常に目標を確認
├── タスク完了 → ワンクリックでDone
├── Confetti演出で小さな達成感
└── 次の目標にフォーカス

夕方〜夜
├── 通知: 「今日の振り返りをしましょう」
├── 各目標の達成/未達成を確認
└── 翌日の準備（任意）

週末
├── 週の振り返り通知
├── 今週の達成状況を確認
└── 来週の3つを設定

月末
├── 月の振り返り
└── 来月の3つを設定
```
